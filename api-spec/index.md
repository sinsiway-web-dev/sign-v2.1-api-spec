# Sign v2.0 API 명세
# 일반
## Header
로그인을 제외한 모든 API는 로그인 성공 시 전달 받은 accessToken을 헤더 Authorization에 명시합니다.
|항목|값(예시)|타입|설명|
|---|---|---|---|
|*Authorization|eyJ0eXAiOiJKV1Q...|token|로그인 성공 인증 토큰|
## Response code
|code|desc|
|---|---|
|200|성공|
|204|결과 없음|
|500|실패|
